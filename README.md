# 🌟 星界神話自動煉金工具

本專案展示如何結合 **PyQt5 GUI、OpenCV 圖像辨識、OCR、滑鼠鍵盤模擬與自動化流程控制**，  
開發出一款能在《星界神話（Astral Realm Online）》中自動執行煉金、領取獎勵的工具。

此專案的重點在於 **影像比對、區域偵測、使用者互動控制與 GUI 整合**，  
展示了完整的自動化控制系統開發能力。

---

## 🧩 功能特色
- 🖼 **自動擷取獎項圖片**：啟動後自動擷取遊戲中所有煉金獎項圖像。  
- 🖱 **互動選取機制**：  
  - 左鍵點擊 → 黑色邊框（僅自動領取一次）  
  - 右鍵點擊 → 紅色邊框（每次抽中皆自動領取）  
- ⚙ **自動煉金流程**：自動點擊「開始煉金」、「領取物品」、「繼續煉金」等操作。  
- 🔍 **即時資源顯示**：透過 OCR 辨識遊戲畫面中「星界碎塊」與「星界石」的剩餘數量。  
- 📌 **視窗固定與快捷鍵控制**：  
  - `F9` → 暫停／繼續  
  - `F8` → 關閉程式  

---

## 🧠 技術棧
- **Python 3.9**  
- **PyQt5** → GUI 介面設計  
- **OpenCV (cv2)** → 圖像辨識與模板比對  
- **PIL (Pillow)** → 圖像處理  
- **ddddocr** → OCR 驗證與文字辨識  
- **pyautogui / keyboard / mouse / pywin32** → 自動控制滑鼠與鍵盤事件
